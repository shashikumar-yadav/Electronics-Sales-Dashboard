# Electronics-Sales-Dashboard

# Project Overview
The Sales Performance Dashboard is an interactive Power BI solution designed to analyze sales data across American regions and electronic product categories. 
It provides a clear view of how individual salespeople contribute to overall performance, enabling managers to track productivity, identify trends, and make informed business decisions.

# Key Features
- Salesperson Analysis: Breaks down sales by individual salesperson, highlighting top performers and areas for improvement.
- Regional Insights: Compares sales across U.S. regions to identify geographic strengths and opportunities.
- Product Category Trends: Analyzes sales by electronic item type, showing demand patterns and profitability.
- Interactive Filters: Allows users to drill down by region, salesperson, or product category for customized insights.
- Performance Tracking: Provides KPIs such as revenue, units sold, and contribution percentage for each salesperson.
  
# Tools & Technologies
- Power BI for dashboard creation and visualization
- Power Query for data cleaning and transformation
- Excel/CSV datasets as the data source
  
# Impact
- Delivered granular visibility into sales performance, helping managers track contributions at both regional and individual levels.
- Supported data-driven decision making by identifying high-performing regions and product categories.
- Improved reporting efficiency by replacing manual spreadsheets with an automated, interactive dashboard.
